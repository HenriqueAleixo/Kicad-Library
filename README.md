# HenriqueAleixo KiCad Repository

Repositório de pacotes customizados do KiCad para instalação via Package Manager.

## Como usar

1. Abra o KiCad
2. Vá em `Plugin and Content Manager`
3. Clique em `Manage...` (canto inferior esquerdo)
4. Adicione este repositório:
   ```
   https://raw.githubusercontent.com/HenriqueAleixo/Kicad-Library/main/repository.json
   ```
5. Salve e instale os pacotes disponíveis

## Pacotes Disponíveis

- **JLCPCB KiCad Library** - Biblioteca completa de componentes JLCPCB/LCSC com símbolos, footprints e modelos 3D
